## 🛠️ Административные возможности

### Через Swagger UI (http://localhost:3001/api/docs)

#### 📦 Управление товарами
- **GET /products** - Просмотр всех товаров
- **POST /products** - Создание нового товара
- **PUT /products/{id}** - Обновление товара
- **DELETE /products/{id}** - Удаление товара
- **GET /products/{id}** - Просмотр конкретного товара

#### 📋 Управление заказами
- **GET /orders** - Просмотр всех заказов
- **POST /orders** - Создание заказа
- **PUT /orders/{id}/status** - Изменение статуса заказа
- **GET /orders/{id}** - Детали заказа

#### 👥 Управление пользователями
- **GET /users** - Просмотр пользователей
- **POST /auth/register** - Регистрация нового пользователя
- **POST /auth/login** - Авторизация
- **PUT /users/{id}/role** - Изменение роли пользователя

#### 📊 Статистика и отчёты
- **GET /admin/stats** - Общая статистика
- **GET /admin/reports** - Генерация отчётов
- **GET /health** - Статус системы
