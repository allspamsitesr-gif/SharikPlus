# 🚀 Быстрый старт SharikPlus

Запустите интернет-магазин воздушных шаров за 5 минут!

## ⚡ Мгновенный запуск

### 1. Клонирование и установка (2 минуты)
```bash
git clone <repository-url>
cd sharikplus-shop
npm install
```

### 2. Запуск всех сервисов (1 минута)
```bash
npm run docker:up
```

### 3. Заполнение данными (1 минута)
```bash
npm run db:seed
```

### 4. Готово! (1 минута)
Откройте в браузере:
- **🎈 Магазин**: http://localhost:3002
- **🛠️ Админка**: http://localhost:3001/api/docs

## 🎯 Что вы получите

### ✅ Полнофункциональный интернет-магазин
- Красивый дизайн с Tailwind CSS
- Каталог из 20+ товаров воздушных шаров
- Адаптивная вёрстка для всех устройств
- SEO оптимизация

### ✅ Мощную админ панель
- Swagger UI для управления API
- Управление товарами (добавление, редактирование, удаление)
- Управление заказами и статусами
- Управление пользователями и ролями
- Статистика и отчёты

### ✅ Современную архитектуру
- **Frontend**: Next.js 14 + React 19 + TypeScript
- **Backend**: NestJS 11 + Prisma + PostgreSQL
- **Кэширование**: Redis
- **Контейнеризация**: Docker

## 🆘 Решение проблем

### Если что-то не работает

#### 1. Порты заняты
```bash
# Найти процессы на портах
lsof -i :3001
lsof -i :3002

# Остановить Docker
docker-compose down
```

#### 2. Проблемы с Docker
```bash
# Очистка
docker system prune -f

# Пересборка
docker-compose build --no-cache
docker-compose up -d
```

#### 3. База данных не работает
```bash
# Пересоздание БД
docker-compose down -v
docker-compose up -d postgres
npm run db:migrate
npm run db:seed
```
