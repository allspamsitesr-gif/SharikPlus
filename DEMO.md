## 🔧 Команды для запуска

### Быстрый старт
```bash
# Запустить все сервисы
docker-compose up -d

# Проверить статус
docker-compose ps

# Посмотреть логи
docker-compose logs -f
```

### Разработка
```bash
# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev

# Сборка
npm run build
```
